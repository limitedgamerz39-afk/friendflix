# Changelog

## [1.1.0] - 2025-11-15
### Added
- Reels/Shorts feature with vertical full-screen video feed
- Watch feature for long-form videos
- Video upload optimization for short-form and long-form content
- New API endpoints for reels and watch feeds
- Video metadata extraction and storage
- Autoplay functionality for video feeds
- Category filtering for watch content
- Migration scripts for existing data

### Changed
- Extended Post and Media models with new fields
- Updated post and media controllers to handle video metadata
- Enhanced media upload screen with long video support
- Improved video player components with autoplay support
- Updated API routes with new endpoints

### Fixed
- Duration extraction for uploaded videos
- Metadata handling for different video types
- Population of video metadata in API responses

## [1.0.0] - 2025-11-01
### Added
- Initial release with core social media features
- User authentication and profile management
- Media upload and sharing
- Feed and search functionality
- Stories feature
- Admin dashboard and monitoring
- Advertisement system
